Getting Started

Overview of the platform and how the three tools connect.

What is the IRC Emergency Platform?
All Tools

The IRC Emergency Platform is a suite of three connected tools that support the Emergency Unit's workflow from crisis detection to funded response:

  1. Emergency Classification — assess and score new emergencies by severity, determining whether IRC should mobilize a White, Yellow, Orange, or Red response.
  2. CRF Allocation Calculator — for Orange and Red emergencies, calculate how much the Crisis Response Fund should allocate based on affected population and coverage targets.
  3. Response Navigator — step-by-step operational guidance for response teams, with tasks organized by phase, sector, and role.

Data flows between the tools automatically. When you classify an emergency, that data is available in the CRF Calculator and Navigator without re-entering it.

What's the typical workflow?
All Tools
  1. Classify the emergency — use the Classification tool to score severity, determine the response stance, and record the classification.
  2. Calculate CRF allocation — if the stance is Orange or Red, open the CRF Calculator. It auto-populates from the classification. Choose a coverage tier (1–3%) and review the funding breakdown.
  3. Navigate the response — open the Response Navigator, select the country, and browse tasks filtered by classification level, response phase, and your role.
You can use any tool independently — they don't require you to follow this order.
What do the response stances mean?
All Tools
  • White — Outside IRC's mission scope. No response required.
  • Yellow — Low severity. Monitor the situation but no CRF funding.
  • Orange — Moderate to high severity. CRF eligible with a ceiling of $1M.
  • Red — Maximum severity. CRF eligible with a ceiling of $2M.

Stances are determined by a matrix combining the emergency's severity score (1–10), the country's pre-crisis vulnerability (1–4), and the response operational burden (0–5).

How do I switch between tools?
All Tools

Every tool has a Site Switcher dropdown in the top-left corner of the header, next to the IRC logo. Click the current tool name to see a dropdown listing all platform apps with their descriptions. Click any app to navigate there.

Emergency Classification

Assess, score, and track emergency classifications.

How do I create a new classification?
Classification
  1. Click Classify in the navigation bar.
  2. Select the emergency type (Conflict, Outbreak, Food Insecurity, Natural Hazard, or Complex).
  3. Choose the country from the dropdown. You'll see the country's vulnerability and robustness scores appear below.
  4. Fill in the emergency details: title, Classification ID, IRC region, processing speed, and dates.
  5. Enter values for the type-specific metrics (e.g., deaths, displaced people). As you type, severity badges update in real time.
  6. Review the calculated result panel showing overall severity and response stance.
  7. Optionally add subnational locations and notes, set the confidence level.
  8. Click Save Classification.
The expiration date auto-calculates to 6 weeks from the notification date if you leave it blank.
How is the severity score calculated?
Classification

Each emergency type has its own set of metrics (e.g., deaths, displaced people for conflict). As you enter values, each metric is scored against historical thresholds on a 1–10 scale. The overall severity is the highest individual metric score.

The severity score then combines with the country's pre-crisis vulnerability (PV, 1–4) and response operational burden (ROB, 0–5) via a matrix to determine the response stance.

How do I reclassify an existing emergency?
Classification

Click on any classification card on the Dashboard to view it, then click Edit. Increase the Reclassification # field and enter the Previous Severity value. Update the metrics with new data.

The system validates reclassification rules: after the first reclassification, severity must generally increase. If your values would violate this rule, a warning banner appears — but you can still save if needed.

Click the reclassification count badge on a dashboard card to see the full timeline of all classification versions.
How do I find specific classifications?
Classification

The Dashboard provides multiple filters that work together:

  • Time period — All, FY26, Last 12 months
  • Stance — All, Red, Orange, Yellow, White
  • Status — Active, Expired, Expiring Soon
  • Emergency type — Conflict, Outbreak, Food Insecurity, Natural Hazards
  • Country search — type a country name to filter
  • Sort by — Date, Severity, or Country

All filters combine — selecting "Red" + "Active" + "Conflict" shows only active red conflict classifications.

How do I use the Maps view?
Classification

Click Maps in the navigation. The map shows countries colored by their highest active stance (red, orange, yellow). You can:

  • Filter by time period, emergency type, stance, region, and status using the sidebar
  • Switch display mode between "Stance color" (categorical) and "Severity heatmap" (gradient)
  • Click a country to see classification details in a popup
  • Use the AI Map Builder — type a natural-language query like "red conflicts in East Africa" and Albert will set the filters for you
  • Export as PNG or SVG, optionally with IRC branding. Edit the title at the top of the page before exporting.
What happens when a classification expires?
Classification

Classifications are valid for 6 weeks from the notification date. As they approach expiration:

  • Within 7 days — an orange warning appears on the card and detail view
  • After expiration — the card shows "EXPIRED" in red. The classification still appears in the dashboard (filterable via the "Expired" status pill) but is excluded from auto-population in the CRF Calculator

To extend a classification, create a reclassification with updated data.

Can I delete a classification?
Classification

Yes. Click a classification card on the Dashboard, then click Delete at the bottom of the detail view. A confirmation dialog will ask you to confirm — this action cannot be undone.

Can I bulk import classifications from a spreadsheet?
Classification

Yes. Click Data in the navigation bar — this takes you to the Platform Admin tool. Drag and drop a CSV or Excel file, map the columns to classification fields using the visual column mapper, preview the data, and import. You can import up to hundreds of records at once.

CRF Allocation Calculator

Calculate Crisis Response Fund allocations for emergency responses.

How do I calculate a CRF allocation?
CRF Calculator
  1. Select a country — type in the search box and choose from the dropdown.
  2. Select a classification (optional) — if active classifications exist for that country, choose one to auto-fill the affected population and stance.
  3. Enter/confirm affected population — the number of people affected by the emergency.
  4. Review the IRC Target — defaults to 10% of affected population. You can edit this.
  5. Confirm the stance — Orange or Red (auto-selected if a classification was chosen).
  6. Choose a coverage tier — 1%, 2%, or 3% of the affected population.
  7. Review the CRF allocation amount and funding gap in the results panel.
Why are some fields auto-filled?
CRF Calculator

When you select a country, the calculator checks for active classifications in the database. If it finds one, selecting it auto-fills:

  • Affected population from the classification's recorded total
  • Response stance (Orange or Red) from the classification
  • Target population calculated as 10% of the affected population

You'll see a yellow "FROM CLASSIFICATION" badge wherever data was auto-populated. You can still manually change any of these values.

What do the coverage tiers mean?
CRF Calculator

The coverage tiers represent what percentage of the total affected population the CRF will fund directly:

  • 1% — Standard initial allocation. Appropriate for most Orange emergencies.
  • 2% — Scaled response for high-severity emergencies or those requiring a larger initial commitment.
  • 3% — Maximum tier, reserved for large-scale emergencies.

The remaining people (gap between CRF coverage and your target) represent the external fundraising needed from donors.

Can I change the target population?
CRF Calculator

Yes. The target defaults to 10% of the affected population (IRC's standard scale target), but you can type any number into the field. The percentage hint updates as you type to show what proportion of affected population you're targeting.

Lowering the target adjusts the total funding needed. If you reduce it below 10%, the Proportional Discount toggle appears (see below).

What is the Proportional Discount?
CRF Calculator

When you manually reduce the target population below 10% of the affected population, a Proportional Discount toggle appears below the results panel.

When enabled, the CRF allocation scales down in proportion to how much you reduced the target. For example:

  • Target at 5% (half of the 10% baseline) → CRF amount is 50% of the standard allocation
  • Target at 3% → CRF amount is 30% of the standard allocation

This ensures the CRF funding ask reflects the stated level of ambition. The toggle is off by default — you can enable it to see the discounted figures, or leave it off for the standard calculation.

What does the ceiling warning mean?
CRF Calculator

If your calculated CRF allocation exceeds the indicative ceiling for the stance ($1M for Orange, $2M for Red), an orange warning appears. This is advisory — it reminds you to consider whether the CRF pool can sustain this alongside other active emergencies. The ceiling is not a hard cap.

How do I compare scenarios?
CRF Calculator
  1. Configure your inputs and review the allocation result.
  2. Click Save Scenario to capture the current state.
  3. Change your inputs (different coverage tier, target, or even country) and save another scenario.
  4. A Scenario Comparison table appears, showing all saved scenarios side by side.

You can save up to 3 scenarios. Each can be individually removed, or use "Clear All" to start over.

How do I share or export a CRF calculation?
CRF Calculator

Click the yellow Copy Summary to Clipboard button. This copies a text summary including the country, classification, CRF allocation amount, coverage tier, and per-client cost to your clipboard. You can paste it directly into an email, Slack message, or document.

How is the allocation amount calculated?
CRF Calculator

The formula is:

CRF Amount = Affected Population × Coverage % × $47.34 per person

$47.34 is the IRC's average direct program cost per person served over 6 months. All amounts are rounded to the nearest $25,000.

The total program cost (dark card at top) uses the target population instead of coverage percentage: Target Population × $47.34.

Response Navigator

Phase-by-phase operational guidance for emergency response teams.

How do I select my role?
Navigator

On the Home page, under "What's your role?", click the pill that matches your function: Response Management, Finance, Logistics/Supply Chain, People & Culture, Safety & Security, Technical Programs, MEAL, Grants, or Partnerships.

Once selected, quick-access links to your key tasks in the immediate response phase (R1) appear below. Your selection is remembered across visits.

How do I filter tasks for a specific emergency?
Navigator
  1. On the Home page, under "Where are you responding?", search for the country.
  2. If active classifications exist, they'll appear — click one to select it.
  3. Click "View Tasks in Navigator" to jump to the task list filtered for that classification level.

The selected classification appears as a colored pill in the header. Click the × on the pill to clear the filter. If no classifications exist for the country, you can select a stance manually (Yellow, Orange, or Red).

How do I find tasks in the Navigator?
Navigator

The Navigator page gives you multiple ways to filter tasks:

  • Sector tabs — click a sector pill at the top (Finance, Logistics, etc.) to see only that sector's tasks
  • Phase timeline — click a response phase (R1 through R6) to filter to that phase
  • Filter bar — search by keyword, filter by classification level, office type, phase, or priority

All filters combine. A results count (e.g., "Showing 14 of 89 tasks") updates live. Click "Clear filters" to reset.

What are the response phases?
Navigator

IRC's emergency response is organized into phases, each with a different focus and timeline:

  • R1 — Immediate Response (first days)
  • R2 — Scaled Response (first weeks)
  • R3 — Sustained Response
  • R4 — Planning & Transition
  • R5 — After Action Review
  • R6 — Ongoing / Preparedness

The phase timeline widget shows each phase with its task count. Click a phase to filter; click it again to deselect.

How do I see task details and linked resources?
Navigator

Click any task card to expand it. You'll see:

  • Subtasks — specific action items within the task
  • Resource links — yellow chips linking to templates, SOPs, or guidance documents (opens in new tab)
  • Cross-references — gray chips linking to related tasks in other sectors

Each task card also shows badges for its phase, priority level, classification applicability, office type, timeline, and owner.

Where do I find templates and documents?
Navigator

Click Resources in the navigation. The Resource Library lists all linked documents across the platform with:

  • Search — find by name or keyword
  • Sector filter — show resources for a specific functional area
  • Link filter — "With Links Only" hides resources without URLs

Each resource shows a file type icon (PDF, DOCX, XLSX, etc.), the context it belongs to, and a "Used by" link that takes you to the relevant task in the Navigator.

Ask Albert (AI Assistant)

AI-powered help available across all three tools.

What is Ask Albert?
All Tools

Albert is the platform's AI assistant, available in every tool. Albert knows about IRC's emergency response protocols, classification rules, CRF funding process, and operational guidance. You can ask in natural language — Albert will figure out what you need.

Click "Ask Albert" in any tool's navigation bar. A chat panel slides in from the right side. Type your question and press Enter.

Albert's answers are context-aware — asking from the CRF Calculator gives CRF-focused responses, while asking from the Navigator gives response-process-focused answers.
What can I ask Albert?
All Tools

Some examples by tool:

  • Classification: "What triggers a reclassification?", "How does the complex emergency rule work?", "Tell me about the Sudan Red classification"
  • CRF Calculator: "How is the CRF allocation calculated?", "What are the Orange vs Red funding ceilings?", "What sections are in the CRF response plan?"
  • Navigator: "What should we do in Week 1 of a Red emergency?", "What does Supply Chain need to do first?", "What's the difference between Orange and Red for Finance?"

Albert also suggests questions based on current active emergencies when you open the panel.

Can I change the AI model?
All Tools

Yes. In the chat panel, a small dropdown above the text input lets you select between available models:

  • Gemini 2.5 Flash (default) — fast, great for most questions
  • Claude Sonnet 4 — better reasoning for complex questions
  • GPT-4.1 Mini — balanced speed and quality

Your model choice is saved and persists across sessions.

Can I ask Albert about a specific task?
Navigator

Yes. In the Navigator, every task card has a small chat bubble icon on the right. Clicking it opens Albert with a pre-filled question about that specific task, including its ID, title, and phase. Albert will give you targeted guidance about completing that task.

How does the AI Map Builder work?
Classification

On the Maps page, a yellow bar below the map says "Build Map." Type a natural-language query like:

  • "Show me red conflicts in East Africa"
  • "Food insecurity emergencies from the last 12 months"
  • "All orange and red in MENA region"

Albert interprets your request and automatically applies the matching sidebar filters. The applied filters show as removable chips in the sidebar — you can fine-tune from there.

How do I give feedback on Albert's answers?
Navigator

In the Navigator's chat panel, each of Albert's responses has thumbs-up and thumbs-down buttons below it. Click one to submit anonymous feedback — this helps improve Albert's responses over time.
